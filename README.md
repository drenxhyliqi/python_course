# Python Course 🐍

This repository contains beginner-level to master-level Python exercises and example scripts to help learn core Python programming concepts.

The goal is to practice Python basics such as variables, loops, conditionals, lists, functions, and more through small, focused scripts.

---

## 📁 What’s Inside

Each file in this repository focuses on a different Python topic:

| File | Description |
|------|-------------|
| `hello.py` | A simple “Hello World” starter script |
| `numbers.py` | Examples working with numbers and arithmetic |
| `logicoperators.py` | Demonstrates logical operators (`and`, `or`, `not`) |
| `conditional.py` | Shows how to use `if`, `elif`, and `else` statements |
| `forloops.py` | Practice with `for` loops |
| `lists.py` | Examples of list creation and manipulation |
| `main.py` | A central file you can use to test and run your own code |

---

## 🧠 Topics Covered

This repository helps you practice:

- Python syntax and structure
- Variables and data types
- Conditional logic
- Looping (`for`, `while`)
- List operations
- Basic input/output
